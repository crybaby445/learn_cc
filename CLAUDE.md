# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## About This Repository

This is a learning environment for experimenting with Claude Code. It uses a devcontainer (Ubuntu 24.04 + Node.js) with Claude CLI installed automatically on startup via `.ona/automations.yaml`.

## Environment

The devcontainer installs Claude Code globally on start:
```
npm install -g @anthropic-ai/claude-code
```

No build, test, or lint commands are configured — this repo is a blank canvas for exploration.
